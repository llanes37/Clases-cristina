Introducción a CSS

CSS, o Hojas de Estilo en Cascada, es el lenguaje que utilizamos para estilizar páginas web. Nos permite definir el diseño visual de los elementos HTML, como colores, tipografías, márgenes, etc.

Colores en CSS

Podemos cambiar los colores de fondo y del texto utilizando las propiedades background-color y color, respectivamente.

Modelo de Caja

El modelo de caja en CSS es fundamental para entender cómo se estructuran los elementos en la página. Cada elemento HTML se trata como una caja que tiene contenido, relleno (padding), borde (border) y margen (margin).
