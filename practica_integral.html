1. Fundamentos de HTML

HTML es el lenguaje base de la web. Sirve para estructurar el contenido de una página. Observa la estructura básica y los comentarios:

<!DOCTYPE html> 
<html lang="es"> 
  <head>
    <meta charset="UTF-8"> 
    <title>Mi Página</title> 
  </head>
  <body>
    <h1>¡Hola Mundo!</h1> 
    <p>Este es un párrafo.</p> 
    
    
    
    
  </body>
</html>
Ejercicios:
  • Completa las partes comentadas en el código anterior.
  • Agrega un segundo párrafo explicando por qué te gusta programar.
  • Pregunta a ChatGPT: "¿Cómo se crea una tabla en HTML?"

2. Introducción a CSS

CSS permite dar estilo a tu HTML. Puedes escribirlo en un archivo externo, en el <head> con <style>, o en línea. Observa el ejemplo y los comentarios:

<style>
body {
  background: #f0f0f0; /* Color de fondo */
  color: #333;          /* Color del texto */
  font-family: Arial, sans-serif; /* Fuente */
  margin: 0;            /* Elimina márgenes por defecto */
}
h1 {
  color: #007bff;       /* Color azul para títulos */
  text-align: center;   /* Centra el texto */
}
a {
  color: green;         /* Color de los enlaces */
  text-decoration: none;/* Sin subrayado */
}
a:hover {
  color: orange;        /* Color al pasar el ratón */
}
/* TU CÓDIGO AQUÍ: Cambia el color de fondo de la tabla y el tamaño de fuente de los párrafos */
</style>
Ejercicios:
  • Personaliza los colores y fuentes de tu página.
  • Haz que los títulos secundarios (<h2>) sean de color rojo y estén en mayúsculas.
  • Pregunta a ChatGPT: "¿Cómo cambio el color de fondo de una tabla en CSS?"
